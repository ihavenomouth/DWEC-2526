Ejercicio 4a2:

Crea una función que reciba cualquier número de parámetros numéricos y devuelva su media. 

const media = (...valores)=>{
  let total = 0;
  for(let i=0; i<valores.length; i++){
    total+=valores[i];
  }

  if( valores.length != 0 )
    return total/valores.length;
  return 0;
}
