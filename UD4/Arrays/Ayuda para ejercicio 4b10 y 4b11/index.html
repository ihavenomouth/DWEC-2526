Ejercicio 4b10

Escribe una función que reciba como entrada un número entero de 5 cifras como máximo y construya una tabla de dos columnas con sus dígitos.

Por ejemplo: 1835 = [1,8,3,5]. A continuación se recorrerá el array escribiendo un número en cada fila de la tabla, la primera columna tendrá el número y la segunda su nombre (uno, ocho, tres, cinco).

Ayuda

La función recibe un entero, habría que comprobar si el número es >=0 y <=99999.

Convertimos el número a cadena de caracteres, si el número es 1831 se puede hacer String(1831). ¿Qué pasa si el número tiene menos de cinco cifras? Se les añaden ceros por la izquierda.

Ahora habría que convertir la cadena a array y cada elemento del array a su equivalente en letras, por ejemplo con un map.

El resto del ejercicio es trivial.

let num = 183;
let numStringuificado = String(num).padStart(5,"0");

numStringuificado.split("").map(n=>{
  if(n=="1") return "uno"
  else if(n=="2") return "dos"
  else if(n=="3") return "tres"
  else if(n=="4") return "cuatro"  
  ...
}) ;
