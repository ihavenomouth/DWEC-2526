Teoría de arrays

Uso básico

Creación, modificación de un elemento, acceso y longitud.

let arr=["uno", "dos", "tres", "cuatro", "cinco"];
arr[4] = 2;
console.log( arr[0], arr[4], arr.length ); // "uno" 2 5

Métodos más interesantes

let arr=["uno", "dos", "tres", "cuatro", "cinco", "tres"];

// ¿Todos los elementos tienen 3 letras?
arr.every(e=>e.length===3); //false

// Encuentra el elemento llamado "dos"
arr.find(e=>e==="dos");

// Devuelve todas las ocurrencias de "tres" en el array
arr.filter(e=>e==="tres");

// Devuelve los elementos que están en las posiciones 1 ("dos"), 2("tres") y 3 "(cuatro)"
arr.slice(1,4);

// Elimina 4 elementos desde la posición 1 y los cambia por "hola"
arr.splice(1,4, "hola"); // queda ["uno", "hola", "tres"];

// Inserta "caracola" justo detrás de "hola"
arr.splice(2,0,"caracola");
