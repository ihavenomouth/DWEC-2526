Ejercicio 5c-1

Ejercicio 1: crea una web que use una hoja de estilos que asigne a los botones y al body estas reglas CSS:

    body{
      background-color: #eee;
      color: #444;
    }
    
    button{
      padding: .5rem;
      margin-bottom: .5rem;
      border-radius: 5px;
      max-width: 100%;
      color: white;
      background-color: #0d47a1;
      border: 1px solid black;
    }

Haz que cuando se pulse un botón con la etiqueta Color, se cambie el color de fondo de los botones a #47adad;

Por otro lado, crea un segundo botón con la etiqueta Modo oscuro/claro que cuando se pulse cambie la hoja de estilos actual a otra hoja de estilo diferente con las reglas:

    body{
      background-color: #456;
      color: #def;
    }
    button{
      padding: .5rem;
      margin-bottom: .5rem;
      border-radius: 5px;
      max-width: 100%;
      color: white;
      background-color: #0d47a1;
      border: 1px solid black;
    }

Si se pulsa el botón de cambiar el modo, se volverá a cargar la hoja de estilos inicial.
