# Instrucciones básicas

Este ejercicio implementa un _backend_ muy básico para poder realizar algunos ejercicios.  También servirá para probar que todo funciona correctamente para el examen.

Para que funciona correctamente necesitas tener NodeJS instalado. Sigue las instrucciones de esta URL si no lo tienes instalado: [https://nodejs.org/en/download](https://nodejs.org/en/download)


Arranca el _backend_ con nodejs ejecutando `npm run start` en la carpeta del proyecto. Si todo ha ido bien aparecerá un mensaje como el de abajo:

> &gt; formypeticiones@1.0.0 start
> &gt; node server.js
>
> Servidor escuchando en http://localhost:3000

Abre en el navegador esa URL ([http://localhost:3000](http://localhost:3000)) y sigue las instrucciones del ejercicio desde ahí.
